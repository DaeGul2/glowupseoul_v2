[0] 사업 개요
--------------------------------------------------------------------------------
강남권 뷰티 시술 큐레이션 컨시어지. 매칭은 입구일 뿐, 사업의 본질은
"상담부터 사후 트래킹까지 풀 저니 케어".

핵심 가치 제안
  ① 시술 정보 탐색 마찰 제거 — 5문항 답하면 AI가 최적 시술/병원 추천
  ② "지인이 데려다주는 느낌" — 1:1 WhatsApp 상담 + 단계별 컨시어지
  ③ 한국 의료관광객의 페인포인트 해소 — 언어/이동/회복기 케어 통합
  ④ 파트너 병원의 진입 마찰 최소화 — 신청 폼만으로 등록


[1] 사업 흐름 (Customer Journey, 6단계)
--------------------------------------------------------------------------------
       ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
       │ 1.상담    │ →  │ 2.확정    │ →  │ 3.입국    │ →  │ 4.병원    │ →  │ 5.종료    │ →  │ 6.사후     │
       │          │    │          │    │          │    │   방문    │    │          │    │   트래킹    │
       └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘

[1-1] 상담 (Consultation)
  - 진입: 메인 앱(얼굴 스캐너 → 폼 → 매칭 결과) → "WhatsApp 상담하기"
  - 운영자가 1:1 응대, 견적/대안 제시
  - 추가 옵션: 화상 영상 상담(Zoom/구글미트), 추가 사진 요청, 보호자 동석 가능

[1-2] 병원·시술·방문날짜 확정 (Booking)
  - 시술 옵션 최종 결정
  - 방문일 캘린더 픽 (병원 가용 슬롯과 연동, 초기엔 운영자 수동 코디)
  - 호텔 추천/예약 대행 (제휴 호텔 또는 외부 부킹 링크)
  - 일정표 발송 (PDF + WhatsApp 메시지)

[1-3] 입국 (Arrival)
  - 공항 픽업 (인천/김포) — 영문 이름판, 차량 등급 선택 (스탠다드/VIP)
  - 의전 동행 (선택) — 입국 게이트부터 호텔/병원까지
  - 환영 키트 (호텔 도착 시) — 시술 안내문, 한국 결제 카드(선택), SIM 카드, 회복 키트

[1-4] 병원 방문 (Treatment Day)
  - 통역 동행 (의료 한·영·중·일)
  - 시술 전 사진/얼굴 스캔 보관
  - 회복 1-2시간 동행, 점심·간식 케어
  - 시술 후 호텔 복귀 안내

[1-5] 종료 (Immediate Aftercare)
  - 처방약/회복 가이드 정리 (다국어)
  - 24시 비상 핫라인 (운영자 휴대폰)
  - 회복 호텔 추천 (조용한 룸/룸서비스)
  - 출국일까지 일일 체크인

[1-6] 사후 트래킹 (Long-term Follow-up)
  - D+7 / D+30 / D+90 회복 체크인 (WhatsApp + 사진 회신)
  - 결과 사진 비교 정리 (Before/After 본인 보관용)
  - 재방문 시 우대 (포인트/할인 코드)
  - NPS 설문 + 후기 요청 (동의 시 익명 후기 노출)


[2] 단계별 부가 서비스 메뉴 (Add-on Catalog)
--------------------------------------------------------------------------------
사용자는 매칭 후 또는 상담 중 옵션으로 추가/제거 가능. 가격은 운영자 견적.

  [공항·이동]
    □ 인천공항 픽업 (스탠다드 세단)
    □ 인천공항 픽업 (VIP — 카니발/제네시스)
    □ 김포공항 픽업
    □ 호텔 ↔ 병원 셔틀 (왕복)
    □ 출국 의전 (체크인·면세 동행)

  [숙박]
    □ 강남 4성/5성 호텔 추천·예약 대행
    □ 회복 전용 호텔 (조용한 객실, 컨시어지 룸서비스)
    □ 장기 체류 (5일+) 할인 패키지

  [의전·통역]
    □ 영어 통역 동행 (시술일)
    □ 중국어/일본어 통역 동행
    □ 보호자 동석 (가족 동행 시)
    □ 시술실 외 대기실 동행

  [회복 케어]
    □ 회복 키트 (압박 마스크/쿨링 패치/약 정리함)
    □ 출장 의료 진료 (붕대 교체/제거)
    □ 사진 동의 후 비포애프터 정리 서비스

  [부가]
    □ K-뷰티 쇼핑 동행 (올리브영/면세점)
    □ K-팝 콘서트/관광지 코디
    □ 포토그래퍼 동행 (회복 후 프로필 촬영)


[3] 사용자 정의 (Personas)
--------------------------------------------------------------------------------
A. 일반 고객 (Customer) — 한국인
   - 20-50대 여성. 시술 처음~중급. 강남 1-2일 코스.

B. 의료관광객 (Medical Tourist) — 외국인
   - 동남아·중화권·미주. 영어 또는 중국어 사용.
   - 4-7일 체류, 호텔/이동/통역 통합 케어 필요.

C. 파트너 병원 (Partner Clinic)
   - 운영자 1인이 폼만 채우면 등록 가능해야 함.


D. 관리자 (Admin)
   - 우리 운영진. 신청 검토, WhatsApp 응대, 컨텐츠 품질, 일정 코디.

[5] 전체 기능 목록
--------------------------------------------------------------------------------

[5-1] 고객 플로우 (오늘까지 구현 완료)
  ① 얼굴 스캐너 (FaceScanner)
     - 카메라 권한 → MediaPipe Face Landmarker 모델 로드 (2-3초)
     - 실시간 478개 랜드마크 추적 (GPU delegate)
     - "Face ID" 풍 골드 메쉬 + 12% 확장 오버레이 + sweep 라인
     - 4.5초 진행 후 video+메쉬 합성 jpeg 캡처 → 결과 페이지 portrait
     - "건너뛰기" 가능 (카메라 거부/미지원 환경)

  ② 선호 입력 폼 (PreferenceForm)
     - 고민 칩 다중선택 19종 (모공/잡티/주름/리프팅/처짐/윤곽/턱선/광대/얼굴축소
       /눈/코/입술/여드름/여드름흉터/피부톤/다크서클/볼륨/탄력/피부결)
     - 예산 슬라이더 (10만 ~ 1,000만원)
     - 다운타임 (0 ~ 21일)
     - 통증 허용도 (1-5)
     - 선호 스타일 (1=자연 ~ 5=드라마틱)
     - 자유 메모 (LLM이 컨텍스트로 해석)

  ③ AI 매칭 결과 (MatchResults + ClinicDetailModal)
     - YOUR SCAN 포트레잇 카드 (얼굴+윤곽)
     - TOP 5 시술 카드: 병원/평점/시술명/이벤트가/원가/할인%/태그/AI 추천 이유
     - "자세히 보기" → 모달 (병원 hero / 특징 / 시술 상세 / AI 추천 이유)
     - "다시 매칭하기" 가능

  ④ WhatsApp 1:1 상담 (Stage 1)
     - 모바일: whatsapp:// 네이티브 스킴으로 즉시 앱 호출
     - 데스크탑: 1.5초 blur 감지 → web.whatsapp.com 폴백
     - prefill 메시지에 병원/시술/가격/다운타임/통증 자동 첨부
     - 운영 번호: +82 10-7386-3249

[5-2] 신규 페이지 (1차 런칭에 추가)
  ⑤ 사업 소개 (/about)
     - 창업 스토리, 철학, 한국 의료관광 통계, 우리가 다른 이유
     - 운영팀 소개 (얼굴/이름 노출 — 신뢰감)

  ⑥ 6단계 저니 시각화 (/how-it-works)
     - 가로 스크롤 또는 아코디언으로 1~6 단계 설명
     - 각 단계별 부가 서비스 카탈로그 인라인
     - 평균 소요시간/주의사항/Q&A 인접 배치

  ⑦ 부가 서비스 카탈로그 (/services)
     - 카테고리: 공항·이동 / 숙박 / 의전·통역 / 회복 케어 / 부가
     - 각 항목 설명 + "이 서비스 문의" → WhatsApp 프리필
     - 가격: "from 5만원~" 표시 또는 "상담"

  ⑧ 병원 디렉토리 (/clinics)
     - 검색 (키워드/지역/카테고리), 필터 (다운타임/통증/스타일)
     - 카드형 그리드, 평점·리뷰 수·시그니처 시술 노출

  ⑨ FAQ (/faq)
     - 의료관광 비자/결제/취소·환불/병원 책임 한계 등

  ⑩ 후기 (/reviews)
     - 동의된 익명 후기 (시술 카테고리 필터)
     - 별점/시술/병원/한 줄 코멘트
     - 비포애프터는 동의된 경우만, 식별정보 마스킹

[5-3] 파트너 플로우 (신규)
  ⑪ 입점 신청 (/partners/apply)
     STEP 1. 병원 기본 정보
       - 병원명, 지역, 주소, 대표 전화
       - 담당자 이름·이메일·휴대폰 (이메일 OTP 인증)
       - 사업자등록번호, 의료기관 등록번호 (필수)
       - 대표 이미지 (선택, 업로드)
       - 한 줄 설명, 강점 키워드 3개
     STEP 2. 시술 등록 (반복 가능, 최소 1개)
       - 시술명, 카테고리
       - 고민 키워드 다중선택
       - 정가/이벤트가 (또는 "상담문의")
       - 다운타임/통증/스타일
       - 한 줄 설명
     STEP 3. 약관 + 제출
       - 정보제공 동의, 의료광고 자율심의 준수 서약
       - 제출 → status='pending'

  ⑫ 신청 현황 조회 (/partners/status)
     - 이메일 + OTP로 현황 확인 (pending/approved/rejected/추가 정보 요청)

[5-4] 어드민 (신규)
  ⑬ 신청 검토 대시보드 (/admin/submissions)
     - 필터 (pending/approved/rejected)
     - 상세: 입력 정보 + 등록증 검토 + 노출 미리보기
     - 액션: 승인 / 반려(사유) / 정보 수정 요청
     - 자동 메일 발송

  ⑭ 저니 운영 (/admin/journeys)
     - 진행 중인 저니 목록 (고객별)
     - 단계 수동 업데이트 (1→6)
     - 부가서비스 발주 체크리스트 (공항 픽업 예약 등)
     - WhatsApp 대화 요약 메모

  ⑮ 컨텐츠 관리 (/admin/clinics, /admin/treatments)
     - 승인된 병원·시술 CRUD
     - 임시 비활성화 (휴진/만료)

  ⑯ 분석 (/admin/analytics)
     - 일별 매칭 시도/완료/WhatsApp 클릭/전환
     - 인기 시술/병원/지역
     - 부가서비스 매출 (단계별 add-on)

[9] 비기능 요구사항
--------------------------------------------------------------------------------
- 모바일 우선 + PWA (홈화면 추가, 오프라인 일부 동작)
- 카메라/MediaPipe는 HTTPS 필수 (PWA 배포 시 자동 충족)
- 신청 폼은 즉시 200 OK + 비동기 OTP/알림 (UX 끊김 방지)

- 의료기관 등록번호는 보건복지부 의료기관 검색 링크 수동 확인
- 개인정보 최소 수집: 매칭 prefs는 익명, 매칭 로그는 60일 후 자동 삭제
- 다국어 (Phase 2): 한/영/중/일 — 외국인 환자 시장

[15] 위험 / 미해결
--------------------------------------------------------------------------------
- 의료광고 심의 통과 못한 시술/카피는 노출 금지 → 카피 작성 단계에서 사전 검토.
- 파트너 가짜 정보 등록 시도 가능 → 의료기관 등록번호 수동 검증 필수.
- 가격 비공개 병원 → "상담문의" 케이스의 매칭 가중치 룰 필요
  (현재 임시: prefilter 통과시키되 가격 정렬에서 후순위).
- WhatsApp 응대 지연 시 신뢰도 하락 → 비즈니스 시간 명시 + 자동 응답 + 알림톡 백업.
- 외국인 환자 유치 시 의료법 개정·비자 이슈 모니터링 필요.
- 부가서비스(픽업/통역) 운영자 직접 수배 vs 외주(타다/지정 통역사) 비용·품질 트레이드.
- 카드 결제 미통합 상태에서 환불 분쟁 발생 시 운영자 수동 처리 부담.
